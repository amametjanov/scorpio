Detailed Description

Algorithms modeled after spmd_utils in the Community Atmosphere Model; C translation.

This includes MPI_Gather, MPI_Gatherv, and MPI_Alltoallw with flow control options.

Provides the functionality of MPI_Alltoallw with flow control options.

Generalized all-to-all communication allowing different datatypes, counts, and displacements for each partner

Parameters
sendbufstarting address of send buffer
sendcountsinteger array equal to the number of tasks in communicator comm (ntasks). It specifies the number of elements to send to each processor
sdisplsinteger array (of length ntasks). Entry j specifies the displacement in bytes (relative to sendbuf) from which to take the outgoing data destined for process j.
sendtypesarray of datatypes (of length ntasks). Entry j specifies the type of data to send to process j.
recvbufaddress of receive buffer.
recvcountsinteger array (of length ntasks) specifying the number of elements that can be received from each processor.
